Drone Project

A showcase of my drone development and engineering experience.

Project Overview

This project demonstrates the design, development, and deployment of a custom drone platform. It covers hardware selection, software integration, and flight testing.

Key Features

  • Custom flight controller integration
  • Real-time telemetry and data logging
  • Autonomous navigation capabilities
  • High-definition video streaming

Technologies Used

  • Arduino / STM32
  • Python, C++
  • ROS (Robot Operating System)
  • OpenCV for computer vision
