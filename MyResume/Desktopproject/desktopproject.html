Project Summary

Our project was based on building an embedded system that utilizes a microcontroller (STM32) to operate an application involving LEDs, ultrasonic sensors, and a display.

Project Components

  • STM32F103C8T6 ("Blue Pill")
  • HC-SR04 Ultrasonic Sensor
  • SSD1306 OLED Display
  • Red & Orange LEDs

System Tasks

  1. Red LED Task
  2. Orange LED Task
  3. Echo Task
  4. Trigger Task
  5. OLED Display Task

FreeRTOS Implementation

We used FreeRTOS to manage our system tasks with priority-based scheduling. Each component's logic was encapsulated in a task. Tasks were synchronized using semaphores and delays using the vTaskDelay function.
